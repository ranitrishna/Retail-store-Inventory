# Retail Sales & Inventory Dashboard

## Project Overview

This project focuses on analyzing retail sales performance and inventory management using Power BI. The dashboard provides insights into sales trends, inventory levels, product performance, and key business metrics to support data-driven decision-making.

## Tools Used

* Power BI
* Excel
* Data Visualization
* Data Analysis

## Key Features

* Sales Performance Analysis
* Inventory Status Monitoring
* Product-wise Performance Tracking
* Interactive Dashboard Visualizations
* KPI Monitoring

## Key Insights

* Identified top-performing products and categories.
* Analyzed sales trends across different periods.
* Monitored inventory levels to support stock management.
* Generated business insights through interactive visualizations.
